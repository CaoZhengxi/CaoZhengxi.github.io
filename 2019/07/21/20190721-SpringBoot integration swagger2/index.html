四、访问SwaggerUI

  • 访问路径http://IP:端口/swagger-ui.html会出现404无法读取到静态资源，控制台显示No mapping found for HTTP request with URI [/swagger-ui.html] in DispatcherServlet with name 'dispatcherServlet'
